# File-Based Grade Analyzer (C++)

A C++ program that reads grade data from a file and computes summary statistics including average, minimum, and maximum values.

## Features
- Reads structured data from a file
- Calculates average, min, and max grades
- Demonstrates file input/output handling
- Efficient processing of datasets

## Concepts Used
- File I/O (ifstream)
- Loops and conditionals
- Basic statistical calculations
